# Learning Path Optimization System

## Project Overview
This project builds a Learning Path Optimization System using Machine Learning. It recommends personalized learning paths for students based on their learning behavior and academic performance.

## Objectives
- Analyze learner behavior
- Recommend personalized learning paths
- Improve course completion rates
- Optimize learning sequences

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Model
- Random Forest Classifier

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Training
6. Prediction
7. Personalized Learning Path Recommendation

## Result
The model predicts suitable learning paths (Beginner, Intermediate, Advanced) based on learner information.
